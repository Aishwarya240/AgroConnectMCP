AgroConnectMCP

Advanced Agricultural Intelligence for Large Language Models

A production-ready Model Context Protocol (MCP) server designed to empower AI with tools for crop disease diagnosis, precise weather forecasting, crop & fertilizer recommendations, and real-time agricultural market news.

Get Started Explore Features

Powerful Agricultural Features

AgroConnectMCP exposes a rich set of capabilities via the Model Context Protocol to enhance farming decisions.

🌿

Plant Disease Diagnosis

Identifies plant diseases and pests from images (URL, local path, or base64) and provides detailed treatment suggestions.

🌾

Crop Recommendation

Recommends optimal crops based on soil nutrients (N, P, K) and environmental factors using a trained Random Forest model.

🧪

Fertilizer Recommendation

Suggests appropriate fertilizers based on soil conditions, crop type, environment, and existing nutrient levels.

🌤️

Weather Intelligence

Comprehensive data including current conditions, 14-day forecasts, soil metrics, evapotranspiration, and UV Index.

🌊

River Discharge

Provides flood risk assessment data via river discharge forecasts powered by Open-Meteo.

📈

Market News

Fetches the latest region-specific agricultural market news and real-time crop price insights.

Prerequisites & API Setup

  • Python 3.12 or higher (if running locally)
  • Docker (if running via container)
  • Kindwise API Key for Crop Health features

Obtaining a Kindwise API Key

The plant disease diagnosis tool relies on the Kindwise Crop Health API.

  1. Visit the Kindwise website: kindwise.com/crop-health
  2. Sign up for an account and navigate to the API section.
  3. Generate a new API key for the Crop Health API.
  4. Keep this key secure. It will be passed as the KINDWISE_API_KEY environment variable.

Installation & Usage

AgroConnectMCP can be run directly via Python or within a Docker container (recommended for production).

1

Build the Docker image

docker build -t agroconnectmcp:1.0.0 .
2

Run the container

Pass the API key as an environment variable. Runs in STDIO mode.

docker run -i --rm -e KINDWISE_API_KEY="your_api_key_here" agroconnectmcp:1.0.0
3

Test with MCP Inspector

npx @modelcontextprotocol/inspector docker run -i --rm -e KINDWISE_API_KEY="your_api_key_here" agroconnectmcp:1.0.0
1

Clone and setup virtual environment

git clone <repository_url>
cd AgroConnectMCP
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
2

Configure Environment

Create a .env file in the project root:

KINDWISE_API_KEY=your_actual_api_key_here
3

Run and Test

Run normally or test with MCP Inspector:

# Direct run
python server.py

# Test with inspector
npx @modelcontextprotocol/inspector python server.py

Configuring MCP Clients

Connect AgroConnectMCP to popular LLM clients like Claude Desktop or LM Studio.

Claude

Claude Desktop

macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
Windows: %APPDATA%\Claude\claude_desktop_config.json 

{
  "mcpServers": {
    "AgroConnectMCP": {
      "command": "docker",
      "args": [
        "run", "-i", "--rm",
        "-e", "KINDWISE_API_KEY=your_api_key",
        "agroconnectmcp:1.0.0"
      ]
    }
  }
}
LM Studio

LM Studio

Add via Developer UI or edit:
~/.cache/lm-studio/mcp/mcp_config.json 

{
  "mcpServers": {
    "AgroConnectMCP": {
      "command": "/absolute/path/to/AgroConnectMCP/.venv/bin/python",
      "args": ["/absolute/path/to/AgroConnectMCP/server.py"],
      "env": {
        "KINDWISE_API_KEY": "your_api_key"
      }
    }
  }
}

Architecture Details

  • Framework: Built using FastMCP for rapid and structured MCP server development.
  • Machine Learning: Utilizes serialized scikit-learn and xgboost models for local inference, ensuring fast responses.
  • External APIs: Integrates with Open-Meteo for comprehensive weather data and DuckDuckGo for market news.
